1629. 按键持续时间最长的键

keys_pressed:  A B C D
release_times: a b c d
求a, b-a, c-b, d-c的最大值对应的key(如果相同，采用字典序最大值)

impl Solution {
    pub fn slowest_key(release_times: Vec<i32>, keys_pressed: String) -> char {
        let keys : Vec<char> = keys_pressed.chars().collect();
        let mut max = release_times[0];
        let mut key = keys[0];
        let l = keys_pressed.len();
        for i in 1..l  {
            let t = release_times[i] - release_times[i - 1];
            if t > max || (max == t && key < keys[i]) {
                max = t;
                key = keys[i];
            }
        }
        key
    }
}

1041. 困于环中的机器人

最棒题解

如果执行一次指令序列后，方向跟初始状态不一致，比如开始为北方，结束为东方，那么下次执行完毕为南方，最后形成一个圈。 如果开始为北方，结束为南方，那么下次执行完毕为北方，又回到原点。

如果执行一次指令序列后，如果回到原点，那么一定走不出去，每次都会回到原点。

class Solution:
    def isRobotBounded(self, instructions: str) -> bool:
        x, y= 0, 0
        # 方向用向量表示，分别代表，N, E, S, W.
        dirs  = [[0,1], [1, 0], [0, -1], [-1, 0]] 
        
        # 初始方向是北
        i = 0 
        for inst in instructions:
            if inst  == 'G':
                # 沿着当前向量移动
                x, y = x + dirs[i%4][0], y + dirs[i%4][1] 
            elif inst == 'L':
                i -= 1 # 逆时针
            elif inst == 'R':
                i += 1 # 顺时针
        
        return (x == 0 and y == 0) or (i%4 != 0)
