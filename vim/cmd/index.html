VIM Cmd

主要依据 spf13/vim

缓冲区

查看缓冲区 :ls

删除缓冲区 :bd buffer delete

跳到指定缓冲区 :b<number>

跳到指定缓冲区 :b<大概文件名>

下一个缓冲区 :bn

上一个缓冲区 :bp

分屏

水平分屏 :split

垂直分屏 :vsplit

分屏切换 <Ctrl>w方向键

tab

tab向后切换 gt

tab向前切换 gT

tab按照索引切换 <0-9>gt

ctrlp

快速检索文件 <Ctrl>p

在检索文件中上下移动 <Ctrl>j <Ctrl>k

退出检索 <Esc>

在新的tab中打开选中文件 <Ctrl>t

在当前tab中垂直split <Ctrl>v

在当前tab中水平split<Ctrl>x

ctrlp翻看历史<Ctrl>p | <Ctrl>n

关闭tab :q

快速搜索

鼠标在某个单词上, 同时按下shift与*按键shift*

tagbar

tagbar toggle <leader>tt

NERDTree

代开/关闭 <Ctrl>e

easyMotion

快速移动 <leader><leader>w

Fugitive git相关操作

查看blame <leader>gb git blame

查看diff <leader>gd git diff

查看status <leader>gs git status
