0x01 Install Dart

参考: https://stackoverflow.com/questions/49286161/how-do-i-install-dart-v2

brew tap dart-lang/dart
brew install dart --devel

0x02 Install Flutter

参考: https://flutter.io/docs/get-started/install/macos

cd ~/development
unzip ~/Downloads/flutter_macos_v1.0.0-stable.zip

# add to path
export PATH="$PATH:`pwd`/flutter/bin"

# follow those tips
flutter doctor

0x03 IDEA

参考: https://flutter.io/docs/get-started/editor?tab=androidstudio

  1. Start Android Studio.
  2. Open plugin preferences (Preferences > Plugins on macOS, File > Settings > Plugins on Windows & Linux).
  3. Select Browse repositories, select the Flutter plugin and click Install.
  4. Click Yes when prompted to install the Dart plugin.
  5. Click Restart when prompted.

0x04 First iOS App

# 开启iOS模拟器
open -a Simulator
flutter new <project name>
cd <project name>
# when flutter run start, press r to refresh
flutter run
