利用hugo, github.io, 搭建个人blog

0x01

在github上创建仓库: <git user name>.github.io

0x02

brew install hugo
# or
go get -u https://github.com/gohugoio/hugo

hugo new site <site name>
cd <site name>
git submodule add <git of the theme you want> themes/<theme name>
# 生成静态文件，在public里。然后把public里的文件推到<git user name>.github.io仓库中
